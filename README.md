# Raspberry Pi Home

## Requirements:

* Maven
* Npm

## How to run
Clone the repository.

```bash
git clone https://github.com/Chameleeon/raspberrypi_home.git
cd raspberrypi_home
```

First you must build the client app:
```bash
cd client
npm run build
```

Run the following commands:

In the server folder:

```bash
./run_server
```
In the client folder:

```bash
./run_client
```
